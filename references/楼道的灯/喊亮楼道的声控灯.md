## 即梦 Seedance 2.0 整体提示词
```
电影级写实风格，5秒短视频。

一只参考@图片1的白猫，背上背着一个浅紫色的兔子形状迷你小书包，
书包侧面挂着一只迷你胡萝卜造型水壶，从傍晚天色渐暗的老小区里，
迈着轻快的肉垫小碎步，走向一栋老式居民楼的楼道入口。

镜头以中景低角度跟随白猫的背影，白猫走入昏暗的楼道，用奶娃音轻叹：“好黑呀”
傍晚的自然余光逐渐被楼道的阴影取代，
画面保持一定对比度，观众能看清暗处白猫的白色毛色和紫色兔子书包。

白猫在黑暗中停下脚步，耳朵轻轻抖了抖，眼睛平视望向楼梯深处，停顿一下
深吸一口气，张开小嘴用奶娃音用力大喊一声“亮！”（发音类似"酿~"），


喊声落下的瞬间，老式声控灯应声亮起，
暖橘色灯光从顶部洒下，照亮整个楼道，
刷着白粉浆的墙面有点脏、地面和墙角的杂物在灯光下显出年代质感，
灯光边缘仍有柔和阴影，形成温暗对比。

镜头微仰捕捉灯光扩散，然后回到白猫面部特写，
白猫眯起圆眼睛，耳朵向后微微一抿，
嘴角上扬露出一个得意又满足的可爱笑容，
尾巴从身后缓缓翘起，尖端轻轻卷曲，仿佛在给自己点赞。

环境音包含傍晚稀疏的虫鸣声、远处居民楼的电视杂音，
以及白猫肉垫轻轻踩在水泥地面上的细微脚步声，
声控灯亮起时伴随轻微的机械"咔嗒"和电流声。
```

## 分段提示词（供参考）

### 0-2.5秒 · 走入楼道

[中景低角度跟随镜头]，
参考@图片1的白猫背着浅紫色兔子造型迷你小书包（侧面挂胡萝卜水壶），
迈着轻快的肉垫小碎步，从傍晚天色渐暗的老小区走向昏暗的楼道入口，
傍晚自然光逐渐过渡到楼道阴影，画面保持对比度确保白猫和紫色书包可见，
背景是老小区斑驳水泥墙面和水泥台阶，
伴随肉垫踩地的细微脚步声、远处居民楼杂音和傍晚稀疏虫鸣。

### 2.5-5秒 · 喊亮灯 + 得意收尾

[微仰镜头 + 面部特写收尾]，
白猫在昏暗楼道中停下，耳朵轻轻抖一抖，
抬头望向天花板老式声控灯，深吸气后张开小嘴，
用奶娃音大喊，参考@音频1的"酿~"猫叫声，
声控灯"咔嗒"亮起，暖橘色灯光从天花板洒满楼道，
斑驳墙面和地面染上温暖质感，灯光边缘保留柔和暗角，
镜头微仰捕捉灯光扩散，再回到白猫面部特写，
白猫眯起圆眼睛、耳朵微抿、嘴角上扬露出得意满足的笑容，
尾巴缓缓翘起，尾尖轻轻卷曲，像给自己点赞。

## 优化亮点

- 🎒 **兔子书包**：浅紫色兔子造型，侧挂胡萝卜水壶，一眼"幼儿园放学"
- 🐾 **耳朵抖一抖**：喊灯前的小迟疑，增加猫的真实萌态
- 💡 **暖橘灯光**：比普通暖黄更温更柔，老楼道也显得有温度
- 😺 **尾巴点赞**：收尾时尾巴翘起卷曲，无需台词也能传达"我超棒的"
- 🔇 **声音分层**：脚步声+虫鸣+远处杂音+灯亮咔嗒，真实感拉满
