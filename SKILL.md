# /jackyshen-long-video-director [-noKeyframeImage]

生成分章节的分镜脚本提示词，用于喂给 seedance 等模型和手工拼接。

- 不带参数：调度所有任务步骤。
- 参数`-noKeyframeImage`：不生成跨章节的首尾帧参考图



## AI视频创作步骤

1. 文案创作
2. 画面分镜
3. 生成首尾帧图
4. 视频生成



### 文案创作
从用户输入的`{主题创意}`生成一篇情节故事，明确立意和要传达的情绪价值。
如果你预计故事较长导致视频生成会超过15秒，或者用户明确要求生成的视频长度要超过15秒，则你要开始拆分多个独立且有逻辑关联的章节，每个章节将来生成的视频长度必须在4-15秒之间。



### 画面分镜
针对上一步故事的每个章节，连同用户输入的参考视频、参考图片、参考音频和特殊要求，调用 `/jackyshen-seedance-promptor` skill 拆分分镜头脚本。输出多个章节的脚本集，每个章节包括多个脚本，每个时间段分镜脚本对应3-5秒的视频画面，每个时间段 都拆成 [运镜] + [主体] + [动作] + [场景] + [氛围/音效] 五字段。
最后再结合整体的立意和要传达的情绪，生成总体的设计要求。



### 生成首尾帧图

为了保证多个章节影片的一致性，如果用户没有指定参考图片作为首尾帧，也没有输入 `-noKeyframeImage` 参数，那么你调用 `/huny-img` 或任何文生图skill为每个章节生成首尾帧图片，命名为“第1章节首帧”、“第2章节首帧兼第3章节首帧”、“第3章节尾帧”。注意，第1章节的尾帧图与第2章节的首帧图必须是同一张图，同样，第2章节的尾帧图与第3章节的首帧图必须是同一张图，以此类推。

### 视频生成
再次调用 `/jackyshen-seedance-promptor` skill，要求将首尾帧图插回到相应的章节分镜脚本中，最后交予用户去手动生成视频。
输出结构的示例参考 `@references/橘猫领导苗乡脱贫致富.md`
